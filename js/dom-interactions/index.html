JavaScript > DOM interactions

Assume you have jQuery or Prototype, write performance-wise script to:

hide all even rows
<script> $(".even").css("display","none"); </script>
do same assuming there are no class names
<script> var hide = false; $("li").each(function() { if (hide) { $(this).css("display","none"); hide = false; } else { hide = true; } }); </script>
Build two arrays containing odd and even rows respectively.
<script> var odd = true; var i_odd = 0; var i_even = 0; var odds = new Array(); var evens = new Array(); $("li").each(function() { if (odd) { odds[i_odd++] = $(this).html(); odd = false; } else { evens[i_even++] = $(this).html(); odd = true; } }); </script>
Rearrange nodes in reverse order
<script> var prev = null; $("li").each(function() { if (prev != null) { $(this).insertBefore(prev); prev = $(this); } else { prev = $(this); } }); </script> OR <script> $("li").each(function() { $(this).prependTo($("ul")); }); </script>

Ability to demonstrate few possible ways to accomplish this is a plus.
