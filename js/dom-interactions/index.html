JavaScript > DOM interactions

Assume you have jQuery or Prototype, write performance-wise script to:

hide all even rows
// using jQuery; I have not worked with Prototype Js before. $('.even').css({'display':'none'}); $('.even').addClass('hideme'); // where '.hideme' is a class with a CSS rule to set the display attribute to none (.hideme {display:none;}) $('.even').hide();
do same assuming there are no class names
// using jQuery $('li:even').css({'display':'none'}); $('li:even').addClass('hideme'); // where '.hideme' is a class with a CSS rule to set the display attribute to none (.hideme {display:none;}) $('li:even').hide();
Build two arrays containing odd and even rows respectively.
/** * Way 1 */ var oddArray = []; var evenArray = []; $('.even').each(function(){ evenArray.push(this); }); $('.odd').each(function(){ oddArray.push(this); }); /** * Way 2 */ var oddArray = []; var evenArray = []; function sortElements (element, arrayToBe){ element.each(function(){ arrayToBe.push(this); }); } sortElements($('.even'),evenArray); sortElements($('.odd'),oddArray);
Rearrange nodes in reverse order
var parentNode = $('ul'); $('li').each(function(){ parentNode.prepend(this); });
