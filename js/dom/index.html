Assume you have only native JS functions available.
Find left position of #d element relatively to the body element
<script> var elem = document.getElementById('d'); var left = elem.offsetLeft + elem.offsetParent.offsetLeft + elem.offsetParent.offsetParent.offsetLeft + elem.offsetParent.offsetParent.offsetParent.offsetLeft; </script>
Modify you code to do same calculation for any nesting level of the #d
<script> var elem = document.getElementById('d'); var left = 0; while( document.body !== elem ) { left += elem.offsetLeft; elem = elem.offsetParent; } </script>
